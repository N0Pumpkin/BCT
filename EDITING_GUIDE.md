## 1. Структура файлов

```
BCT/
├── index.html          ← Главная страница
├── catalog.html        ← Страница каталога (все категории)
├── services.html       ← Страница услуг
├── delivery.html       ← Страница доставки и оплаты
├── contact.html        ← Страница контактов
│
├── catalog/            ← Отдельные страницы категорий (у каждой свой URL)
│   ├── roofing.html    ← Кровля
│   ├── facade.html     ← Фасады
│   ├── insulation.html ← Изоляция
│   └── drainage.html   ← Водосток
│
├── css/
│   └── style.css       ← Все стили сайта (цвета, шрифты, отступы)
│
├── js/
│   ├── data.js         ← ВСЕ ДАННЫЕ САЙТА (товары, контакты, тексты)
│   ├── components.js   ← Хедер, футер, модальное окно
│   └── script.js       ← Логика (слайдеры, меню, рендер)
│
├── img/                ← Все изображения
│   ├── main/           ← Фото для слайдера и категорий (главная)
│   ├── roof/           ← Фото кровельных товаров
│   ├── facade/         ← Фото фасадных товаров
│   ├── insul/          ← Фото изоляции
│   └── drain/          ← Фото водостока
│
└── logo/
    ├── logo.png        ← Логотип компании (отображается в шапке)
    ├── logo1.png       ← Запасные варианты логотипа
    └── logo2.png
```

> **Главное правило:** Почти весь контент сайта находится в файле `js/data.js`.
> Редактируйте именно его, остальные файлы трогать не нужно.

> **Как открыть сайт локально для проверки:**
> Дважды кликните по `index.html` — откроется в браузере. Либо запустите
> локальный сервер из папки проекта: `python3 -m http.server 8000`
> и откройте `http://localhost:8000` в браузере.

## 2. Как изменить контакты

Откройте файл `js/data.js`. В самом начале найдите раздел `contacts`:

```javascript
contacts: {
  phone:       "+77773377701",            // ← Телефон (без пробелов)
  phoneDisplay:"8 (777) 33 777 01",       // ← Телефон как он показывается
  email:       "info@buildingcitytime.kz",// ← ОСНОВНОЙ email
  emails: [                               // ← Список email отделов
    { label: "Общие вопросы", addr: "info@buildingcitytime.kz"  },
    { label: "Бухгалтерия",   addr: "buh@buildingcitytime.kz"   },
    { label: "Закуп",         addr: "zakup@buildingcitytime.kz" },
    { label: "Отдел продаж",  addr: "sales@buildingcitytime.kz" },
  ],
  whatsapp:    "https://wa.me/77773377701", // ← Ссылка на WhatsApp
  address:     "г. Астана, ул. Достык 16, Talan Towers", // ← Адрес
  mapEmbed:    "https://www.google.com/maps/embed?...", // ← Ссылка на карту
},
```

**Как работают два поля email:**
- `email` — основной адрес. Показывается в шапке, футере и в кнопках
  «Написать на почту» по всему сайту. Меняете один раз — меняется везде.
- `emails` — полный список отделов. Показывается **только** в блоке
  «Реквизиты» на странице «Контакты». Чтобы добавить/убрать отдел —
  добавьте или удалите строку `{ label: "...", addr: "..." }`.

**Как получить новую ссылку на карту Google:**
1. Откройте Google Maps, найдите нужный адрес
2. Нажмите «Поделиться» → «Встроить карту»
3. Скопируйте ссылку из атрибута `src="..."` в коде iframe
4. Вставьте её в поле `mapEmbed`

После сохранения — контакты автоматически обновятся на всех страницах сайта.

## 3. Как добавить товар в каталог

Откройте `js/data.js`. Найдите нужную категорию в массиве `catalog`, например `"Кровля"`:

```javascript
{
  id:    "roofing",
  label: "Кровля",
  items: [
    // Здесь товары — скопируйте один из блоков ниже
  ],
},
```

**Шаблон нового товара (скопируйте и заполните):**

```javascript
{
  id:     "my-product",           // Уникальный ID (латиница, без пробелов)
  name:   "Название товара",      // Отображаемое название
  sub:    "Подзаголовок",         // Краткое описание / тип
  images: [
    "img/roof/photo1.jpeg",       // Путь к первому фото (обязательно)
    "img/roof/photo2.jpeg",       // Второе фото (если есть)
  ],
  specs: [
    { label: "Толщина",    value: "50 – 100 мм" },
    { label: "Размер",     value: "1200 × 600 мм" },
    { label: "Материал",   value: "Сталь" },
    // Добавляйте сколько нужно характеристик
  ],
  application: "Для чего применяется этот материал.",
  objects:     "Для каких объектов подходит.",
  price:       "от 1 500 ₸/м²",  // Цена или "Цена по запросу"
  hasTDS:      false,             // true — если есть кнопка "Скачать ТДС"
},
```

**Важно:** каждый товар в массиве `items` должен заканчиваться запятой (`,`),
кроме последнего.

## 4. Как добавить фото к товару

> **Где живут все фото:** в папке `img/` в корне сайта.
> Все фото с прошлого сайта уже на месте — посмотрите её структуру,
> прежде чем что-то добавлять.

1. **Загрузите фото** в нужную папку внутри `img/`:
   - кровельные товары → `img/roof/`
   - фасадные → `img/facade/`
   - изоляция → `img/insul/`
   - водосток → `img/drain/`
   - картинки для главной/слайдера → `img/main/`

2. **Имя файла:**
   - используйте только латиницу, цифры, дефис и нижнее подчёркивание
   - старайтесь обходиться без пробелов и кириллицы
   - пример хорошего имени: `metal-sheet-blue.jpeg`
   - если в имени уже есть пробел (как у `panels (1).jpeg`) — это работает,
     но в `data.js` путь надо записывать ровно как имя файла:
     `"img/facade/panels (1).jpeg"`

3. **Формат и размер:**
   - `.jpeg`, `.jpg`, `.png` или `.webp`
   - рекомендуемый размер: **800×600 px** или пропорция **4:3**
   - вес одного фото — не больше 300–500 КБ (можно сжать на [tinypng.com](https://tinypng.com))

4. **Укажите путь к фото** в поле `images` товара в `js/data.js`:

```javascript
images: [
  "img/roof/new-photo.jpeg",
],
```

Если добавите несколько фото — они автоматически станут слайдером на карточке:

```javascript
images: [
  "img/roof/photo-1.jpeg",
  "img/roof/photo-2.jpeg",
  "img/roof/photo-3.jpeg",
],
```

> **Совет:** первое фото в массиве будет показано на карточке по умолчанию —
> выбирайте самое выразительное.

## 5. Как изменить цену товара

В `js/data.js` найдите нужный товар и измените поле `price`:

```javascript
price: "от 2 500 ₸/м²",
// или
price: "Цена по запросу",
// или
price: "1 200 ₸/шт",
```

Если поле оставить пустым (`price: ""`), цена не отобразится.

## 6. Как добавить новую категорию

В `js/data.js` найдите массив `catalog` и добавьте новый объект:

```javascript
{
  id:    "windows",                      // Уникальный ID (латиница)
  slug:  "catalog/windows.html",         // Путь к странице категории
  label: "Окна",                         // Название в меню/карточках
  desc:  "Металлопластиковые окна",      // Краткое описание
  image: "img/main/windows.jpg",         // Фото категории (для превью)
  items: [
    // Товары — см. шаблон в разделе 3
  ],
},
```

Затем создайте HTML-файл для страницы категории:
1. Скопируйте файл `catalog/roofing.html`
2. Переименуйте в `catalog/windows.html`
3. Откройте и измените строку:
   ```html
   <body data-page="category" data-category="roofing">
   ```
   на:
   ```html
   <body data-page="category" data-category="windows">
   ```
   Здесь `windows` должно совпадать с `id` категории в `data.js`.
4. Обновите `<title>` и `<meta name="description">` под новую категорию.
5. Положите фото категории в `img/main/` (как указано в `image`).

> **Готово.** Содержимое страницы (заголовок, описание, товары) подтянется автоматически из `data.js`. Менять `script.js` или `components.js` не нужно.

## 7. Как изменить цвета на сайте

Откройте `css/style.css`. В самом начале файла — раздел **«1. ПЕРЕМЕННЫЕ»**:

```css
:root {
  /* --- Основные цвета --- */
  --color-bg:         #0B0B0B;   /* фон всего сайта */
  --color-accent:     #F4E831;   /* жёлтый акцент (кнопки, рамки, иконки) */
  --color-accent-dark:#D4C920;   /* жёлтый при наведении */
  --color-white:      #FFFFFF;
  --color-text:       #EDEDED;   /* основной текст */
  --color-text-muted: #B9BFC6;   /* вторичный текст */
  ...
}
```

**Популярные изменения:**

| Что изменить | Переменная |
|---|---|
| Цвет кнопок | `--color-accent` |
| Фон сайта | `--color-bg` |
| Цвет текста | `--color-text` |
| Цвет второстепенного текста | `--color-text-muted` |
| Фон карточек | `--color-surface-2` |
| Цвет рамок | `--color-surface-3` |

Цвета указываются в формате HEX (`#FF0000`). Используйте сервис
[colorpicker.me](https://colorpicker.me) для выбора цвета.

## 8. Как изменить текст на главной

**Заголовок и подзаголовок hero:**
Откройте `index.html`, найдите секцию `hero`:

```html
<div class="hero-kicker">Строительные материалы<br>и монтаж под ключ</div>
<div class="hero-title">Кровля · Фасады · Изоляция · Водосток</div>
<p class="muted">Прямые поставки от производителей...</p>
```

**Слайдер hero (подписи к фото):**
В `js/data.js` найдите `heroSlides`:

```javascript
heroSlides: [
  { src: "img/main/photo.png", title: "Заголовок слайда", sub: "Описание слайда" },
  ...
],
```

**Преимущества на главной:**
В `js/data.js` найдите `advantages`:

```javascript
advantages: [
  { icon: "🏗️", title: "Заголовок", text: "Описание преимущества." },
  ...
],
```

## 10. Как изменить кнопку WhatsApp

Кнопка WhatsApp (плавающая, в правом нижнем углу) ведёт на номер из `contacts.whatsapp` в `data.js`.

**Чтобы изменить позицию кнопки** — откройте `css/style.css`,
найдите раздел **«20. WHATSAPP»**:

```css
.whatsapp-fab {
  right: 20px;    /* ← отступ от правого края */
  bottom: 24px;   /* ← отступ от нижнего края */
  ...
}
```

**Чтобы убрать пульсацию:**
Найдите `animation: wa-pulse` и удалите эту строку.

## 11. Часто задаваемые вопросы

**Q: Изменил data.js, но ничего не изменилось.**
A: Обновите страницу в браузере (Ctrl+F5 или Cmd+Shift+R). Иногда браузер кэширует старый файл.

**Q: Как изменить логотип?**
A: Замените файл `logo/logo.png` на свой файл с тем же именем. Если новый
логотип получился слишком большим или маленьким — в `css/style.css`
найдите раздел **«6. ХЕДЕР»** и поменяйте `--logo-height` (по умолчанию `40px`)
или `max-width: 180px` для очень широких логотипов.

**Q: Как убрать кнопку «Скачать ТДС» у товара?**
A: Установите `hasTDS: false` в настройках товара в `data.js`.

**Q: Где взять ссылку для WhatsApp?**
A: Формат: `https://wa.me/7XXXXXXXXXX` (без пробелов, с кодом страны).

**Q: Как изменить текст в кнопках навигации?**
A: В `js/data.js` найдите массив `nav`:
```javascript
nav: [
  { label: "О компании",  href: "index.html" },
  { label: "Каталог",     href: "catalog.html" },
  ...
],
```
Измените `label`.

**Q: Как поменять порядок товаров в категории?**
A: Перетащите блок товара выше/ниже в массиве `items` в `data.js`. Порядок в коде = порядок на сайте.

**Q: Как добавить строку в таблицу характеристик товара?**
A: В массиве `specs` нужного товара добавьте строку:
```javascript
{ label: "Гарантия", value: "5 лет" },
```
